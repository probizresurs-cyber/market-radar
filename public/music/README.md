# Библиотека фоновой музыки

Используется конвейером «Контент-завод → видео» (`/api/content/video/render`,
см. `src/lib/music-library.ts`) для подбора фоновой музыки по настроению
ролика. Без файлов здесь — ролики просто рендерятся без музыки (только
голос + субтитры), это штатное поведение, не ошибка.

**Я (агент) сознательно не скачивал сюда никакие треки сам** — лицензию
любого файла, который я бы нашёл и положил сюда без вашего ведома, я не
могу проверить и поручиться за неё, а публиковать в клиентских видео
музыку с неясными правами — риск. Это то, что должен сделать человек,
выбрав треки осознанно.

## Как добавить треки

1. Скачайте несколько (10-20) коротких (30-90 сек, либо зацикливаемых)
   инструментальных треков с чёткой royalty-free/CC0 лицензией. Источники:
   - **Pixabay Music** — https://pixabay.com/music/ (полностью бесплатно,
     без атрибуции, фильтр по настроению есть на сайте)
   - **YouTube Audio Library** — https://studio.youtube.com → Audio Library
     (нужен любой Google-аккаунт, фильтр по mood/genre/instrument)
   - **Free Music Archive** — https://freemusicarchive.org (проверяйте
     лицензию каждого трека — там смешаны CC0/CC-BY/CC-BY-NC)
2. Положите файлы `.mp3` прямо в эту папку (`public/music/`).
3. Опишите каждый в `manifest.json` — массив объектов:

```json
[
  { "file": "corporate-upbeat.mp3", "moods": ["corporate", "upbeat"], "license": "CC0 — Pixabay" },
  { "file": "calm-piano.mp3", "moods": ["calm", "dramatic"], "license": "CC0 — Pixabay" }
]
```

`moods` — свободные теги, режиссёр-агент присваивает ролику один из:
`upbeat`, `calm`, `corporate`, `dramatic`, `playful`. Если ни один трек не
подходит под настроение — библиотека тихо отдаёт любой случайный (или
`null`, если файлов вообще нет), рендер не падает в любом случае.
